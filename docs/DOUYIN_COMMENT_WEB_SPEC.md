# 抖音自动评论WEB页面 - 需求文档 v2.0

## 一、背景与问题

### 现有问题
1. 账号管理页面不显示登录状态（已登录/未登录/过期）
2. 创建任务时选择"平台"，逻辑错误——应选择"账号"
3. 自动化规则参数完全缺失

### 参考竞品
竞品RPA软件包含完整的节奏控制、过滤筛选、互动行为配置。

## 二、页面结构

### 2.1 账号管理页面改造
在现有账号列表增加"登录状态"列：
- 状态：已登录(绿) / 未登录(灰) / 登录过期(红)
- 最后登录时间
- 操作：检测登录 / 重新登录

### 2.2 抖音评论页面（新增）

#### 区块1：账号选择
- 下拉选择已登录的抖音账号（只显示已登录的）
- 显示账号昵称、登录状态

#### 区块2：搜索配置
| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 搜索关键词 | textarea | - | 每行一个关键词，执行时轮换 |
| 时间筛选 | select | 不限 | 不限/一天内/一周内/一个月内 |
| 排序方式 | select | 综合排序 | 综合/最新/最热 |

#### 区块3：节奏控制
| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 评论视频数量 | number | 10 | 总共评论几个视频 |
| 评论间隔(秒) | range输入 | 1-3 | 格式：最小值-最大值 |
| 评论几个换关键词 | number | 10 | 每N个视频切换下一个关键词 |
| 发送几个休息一下 | number | 5 | 每N次操作触发休息 |
| 休息间隔(秒) | range输入 | 5-10 | 格式：最小值-最大值 |

#### 区块4：评论内容
| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 评论内容 | textarea | - | 每行一条评论，随机抽取 |
| 是否@用户 | checkbox+input | 未勾选 | 勾选后填@用户名，如：豆包/风启 |
| 是否发图片 | checkbox+input | 未勾选 | 勾选后填图片文件夹路径 |

#### 区块5：过滤筛选
| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 过滤省份 | checkbox+multi-select | 未勾选 | 勾选后多选省份，排除这些地区的用户 |
| 过滤时间 | checkbox+select | 未勾选 | 勾选后选择时间范围 |
| 评论区关键词筛选 | checkbox+input | 未勾选 | 勾选后输入筛选关键词（空格分隔） |
| 筛选数据量 | number | 10 | 关键词筛选时处理几个用户 |

#### 区块6：互动行为
| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 点赞 | checkbox | 勾选 | 模拟点赞行为 |
| 关注 | checkbox | 勾选 | 模拟关注行为 |
| 收藏 | checkbox | 勾选 | 模拟收藏行为 |
| 浏览 | checkbox | 勾选 | 模拟浏览行为（养号） |

#### 区块7：执行控制
- 按钮：开始执行 / 暂停 / 继续 / 结束
- 显示：已执行任务数、当前状态

#### 区块8：执行结果
- 实时日志输出
- 截图证据展示

#### 区块9：执行历史
- localStorage存储
- 字段：时间/关键词/视频标题/评论/状态/耗时

## 三、后端API

### POST /api/douyin-comment
请求体：
{
    "account_id": 1,
    "keywords": ["香港移民", "移民"],
    "time_range": 30,
    "sort_by": "general",
    "video_count": 10,
    "comment_interval": [1, 3],
    "keyword_rotate_after": 10,
    "rest_after": 5,
    "rest_interval": [5, 10],
    "comments": ["评论1", "评论2"],
    "mention_user": "",
    "send_image": false,
    "image_folder": "",
    "filter_province": [],
    "filter_time": 0,
    "filter_keywords": [],
    "filter_count": 10,
    "actions": {"like": true, "follow": true, "favorite": true, "view": true}
}

### POST /api/douyin-comment/control
暂停/继续/结束控制

## 四、前端技术要求
- 风格与overview.html一致（暗色主题Catppuccin Mocha）
- 侧边栏导航与现有页面一致
- 区块可折叠/展开
- range输入支持双值（最小-最大）

## 五、验收标准
1. 账号管理显示登录状态
2. 创建任务选择账号而非平台
3. 所有自动化规则参数可配置
4. 执行调用OpenClaw赵云agent
5. 实时日志、截图证据、历史记录正常
6. 风格与现有页面一致
