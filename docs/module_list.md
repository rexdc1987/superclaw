# SuperClaw MVP Module List

1. Account Manager - multi-account, groups, health status
2. Keyword Library - groups, rotation, include/exclude
3. Comment Collector - search, filter, dedup
4. Lead Engine - scoring, assignment, tracking
5. Action Runner - comment, reply, like, follow, DM
6. Risk Control - rate limits, cooldown, circuit breaker
7. Material Service - templates, variables, sensitive words
8. Task Center - state machine, pause/resume/retry
9. Audit Log - operation, task, business, exception
10. Data Export - CSV/XLSX, dashboard

Data Models: Account, AccountGroup, KeywordGroup, Task,
TaskRule, Comment, Lead, Action, MessageTemplate,
Material, RiskRule, ExecutionLog, AuditLog
